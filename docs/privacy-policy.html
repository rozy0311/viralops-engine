1. Introduction

ViralOps ("we", "us", "our") operates the ViralOps Engine platform ("the Service"). This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our Service.

We are committed to protecting your privacy and handling your data transparently. Please read this policy carefully. By using the Service, you consent to the data practices described in this Privacy Policy.

2. Information We Collect

2.1 Information You Provide

  • Account Information: Email address, username, and profile information when you create an account

  • Content Data: Text, images, videos, and other media you create or upload through the Service
  • Configuration Data: Scheduling preferences, posting templates, hashtag strategies, and account settings

2.2 Information Collected Automatically

  • Usage Data: Features used, actions performed, posting history, and scheduling patterns
  • Analytics Data: Engagement metrics (likes, comments, shares, views) retrieved from connected platforms
  • Technical Data: IP address, browser type, device information, and access timestamps
  • Log Data: Server logs, error reports, and performance metrics

2.3 Information from Third-Party Platforms

When you connect social media accounts, we may receive:

  • Public profile information (name, username, profile picture)
  • Account statistics and analytics data
  • Content performance metrics
  • Follower/following counts and engagement rates

4. Data Storage & Security

We implement appropriate technical and organizational measures to protect your data:

  • Encryption: All API keys and OAuth tokens are stored securely using environment variables, never in plaintext
  • Access Control: Strict authentication and authorization controls
  • Local Processing: Content processing and scheduling occur on your infrastructure when self-hosted
  • Database Security: SQLite databases are stored locally with appropriate file-system permissions
  • No Credential Logging: Sensitive credentials are never written to log files

While we strive to use commercially acceptable means to protect your data, no method of transmission or storage is 100% secure.
